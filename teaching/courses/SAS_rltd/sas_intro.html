Exercises for an easy start with Sas

  • (Exercise 1) Using an introductory SAS manual(which I highly recommend you to purchase), learn how to create SAS datasets, and do statistical analysis. First exercise is to create a small sas program that can input data on three variables, y, x1, x2 (you provide some arbitrary numbers for those variables) and then print the dataset, and also run a regression of y on x1, x2. Here is a good starting point sas example. In this example you will see how to create temporary sas datasets, merge two datasets, get descriptive statistics, run regressions, obtain frequency distribution. These are explained in the source code of the program file example1.sas. How to run sas under windows or unix is not explained here. For a really short introduction that I wrote some time ago about how to run SAS under unix follow this url: click here for a really short introduction to SAS under unix by L.K. Raut
